# Auto Logging: Deforestation Simulator

Stop the manual grind and master the wilderness with Auto Logging: Deforestation Simulator. This mod streamlines the process of clearing land, processing timber, and organizing your resources through automated area-selection tools.

**This update brings quite a few new features and may require an uninstall/reinstall of the mod.**

## Compatibility and Requirements
The current target version of this mod is 42.13+ including 42.14. I won't be actively developing for any versions before that.
- Build 41: Supports both Singleplayer and Multiplayer.
- Build 42.12: Supports Singleplayer only.
- Build 42.13: Supports Singleplayer and Multiplayer; Requires JB ASSUtils.
- Build 42.14: Supports Singleplayer and Multiplayer; Requires JB ASSUtils.

## Core Features

**Automated Gathering**
Efficiency is key to survival. Simply designate your desired storage location and select a target area to automatically gather the following:
- Logs
- Planks
- Firewood
- Twigs and Branches

**Land Clearing**
Clear out large swaths of land with ease. Use the area selection tool to:
- Fell trees
- Remove bushes
- Cut down grass and weeds

**Timber Processing**
Turn your raw timber into usable building materials or fuel. You can select an area of logs to automatically:
- Saw logs into planks
- Chop logs into firewood

## Advanced Storage Logic
This mod introduces Exclusive Storage tiles designed specifically for Logs, Planks, Scrap Wood, Twigs, Branches, and Firewood. These specialized storage areas have a capacity of 100. "R" to rotate, RMB / Remove Storage to remove it.

The gathering system is highly adaptive to your current base setup:
- Exclusive Storage: Items will fill the designated tile and then automatically move to adjacent storage areas of the same type until all are full.
- Containers and Vehicles: You can designate crates, stacks of crates, or vehicle trunks/truck beds as storage destinations. Stacks of crates will fill vertically.
- Overflow Handling: If a container or truck bed reaches capacity, the mod defaults to dropping remaining items on the ground near the container or the rear driver-side door of the vehicle.
- Open Ground: You may also choose any empty tile to drop gathered items in a pile.

## Customization Options
Tailor the interface to your playstyle through the Mod Options menu:
- Interface: Toggle the sidebar menu on or off under the "Place" tool menu.
- Menu Behavior: Set the Logging Mouse Menu to always show or only appear when actions are available.
- Menu Positioning: Choose to have the logging options appear at the top of the right-click menu or after the "Sit on Ground" option.
- Visuals: Customize the color of the selection tool outline for better visibility.

## Limitations
